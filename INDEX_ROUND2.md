## Key Findings

### What Works
- **Hammer (WORK):** Bold, monolithic, politically meaningful. Hammers = labor/solidarity. Fills space well. Reads clearly at 28x28px.

### What Fails
- **House (PESSOAL):** Generic utility shape. Appears in every design system. No political meaning. Window cutout is ornamental fluff.
- **Star+Sparks (PROJECTS):** Suffers from "decorative ornament disease." Spark triangles break the monolithic form. Feels like generic tech startup "innovation" icon, not a political movement graphic.

### The Critical Insight
The problem isn't technical execution. Both failing icons are technically sound (organic coords, filled, readable). The problem is **PERSONALITY and MEANING**.

SOS hand and PCP star carry weight because they're symbols of power and solidarity. The icons must evoke similar gravity — they should feel at home on a protest banner or revolutionary flag.

## Validation Frameworks Created

### 1. Visual Gravity Test
**Question:** Would this design appear on a protest banner or revolutionary flag?
- ✓ YES = Icon has personality, meaning, gravity
- ✗ NO = Icon is too generic or too ornamental

**Applied to:**
- SOS hand → YES (solidarity symbol)
- PCP star → YES (revolutionary power)
- Hammer → YES (labor/solidarity)
- House → NO (generic)
- Star+Sparks → NO (decorative)

### 2. Decoration Test
Check that icons are monolithic (no ornamental details):
- ✗ NO spark lines
- ✗ NO accent shapes
- ✓ YES bold forms only

### 3. Boldness Test
Icons must fill 60-75% of viewBox:
- ✓ Hammer = good proportion
- ✗ House = timid, too small
- ✗ Star+Sparks = medium, but sparks break unity

### 4. Code Review Checklist
For Coder validation:
- Organic asymmetric coordinates ✓
- Zero strokes anywhere ✓
- Readability at 28x28px ✓
- Hand-traced bezier feel ✓

## Key Takeaways for Round 3+

### Critical Principles
1. **Monolithic Forms Only** — No decoration, no sparkles, just bold shapes
2. **Fill 60-75% of viewBox** — Shapes dominate, don't hide in corners
3. **Political Meaning** — Each icon must carry weight (hammer = labor, leaf = growth, lightning = action)
4. **The Banner Test** — "Would this work on a flag?" Must answer YES
5. **Hand-Traced Feel** — Organic coords, asymmetric bezier, never perfect geometry

### No More Generic Icons
- SOS hand is a gesture with MEANING (solidarity)
- PCP star is a symbol of POWER (revolution)
- Every icon must echo that same gravity
- If it could be a Font Awesome icon painted red, it's not bold enough
