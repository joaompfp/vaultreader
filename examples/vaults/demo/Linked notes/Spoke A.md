# Spoke A

A small note that links back to [[Hub]] and across to [[Spoke B]].

The graph view will draw a triangle: Hub ↔ Spoke A ↔ Spoke B ↔ Hub.

> [!info] Spoke notes
> The three spokes (A, B, C) deliberately link to each other to make the graph view interesting. Without these cross-links, the graph would just be a star.
