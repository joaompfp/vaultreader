- House is ubiquitous in app design (every software has a "house" icon)
- Zero political meaning or personality
- Doesn't answer: "What does this symbol mean?"
- Window cutout is decorative fluff
- Would NOT work on a protest banner
- Scored **3/10** on visual gravity test

**Why It Failed:**
```
Visual Gravity Test (key question): 
"Would this appear on a protest banner, labor poster, or communist party flag?"
Answer: NO ❌ — Generic, corporate, boring
```

- Leaf = GROWTH, personal development, LIFE (meaningful)
- Organic curved form matches SOS Racismo hand aesthetic
- Single unified path with internal vein structure (visual interest)
- Fills 70% of viewBox (visual gravity, bold presence)
- Would work on a solidarity/environmental poster
- Scores **7/10+** on visual gravity test (estimated)

**Why It Works:**
```
Visual Gravity Test (key question):
"Would this appear on a protest banner, labor poster, or communist party flag?"
Answer: YES ✅ — Growth, life, personal development are politically meaningful
```

## Icon 3: PROJECTS — Star+Sparks vs. Lightning Bolt

### Round 2: Star with Spark Lines ❌

**Problem:** Decorative ornament disease
- Star alone is okay (PCP reference), but sparks destroy the form
- Spark lines are decorative fluff (not bold forms)
- Violates monolithic principle (2 separate paths = visual noise)
- Makes the icon feel like generic "innovation" (Font Awesome style)
- Would NOT work on a political banner
- Scored **2/10** on visual gravity test

**Why It Failed:**
```
Visual Gravity Test:
"Would this appear on a protest banner or revolutionary flag?"
Answer: NO ❌ — Feels like a SaaS startup icon, not political graphics

Monolithic Form Check:
"Is this a single unified shape?"
Answer: NO ❌ — Star + sparks are separate decorative elements
```

**Detailed Critique (from Round 2):**
> "I tried to differentiate from PCP star by adding spark lines. But this is DECORATIVE ORNAMENT DISEASE — adding tiny details instead of boldness. SOS hand doesn't have spark lines. PCP star doesn't have decorative sparkles. These logos are MONOLITHIC FORMS, not decorated shapes."

### Round 3: Monolithic Lightning Bolt ✅

**Solution:** Single bold zigzag form
- Lightning = SPEED, ENERGY, ACTION, INNOVATION (meaningful)
- Monolithic single path (zero separate shapes)
- Bold zigzag fills 75% of viewBox vertically (visual gravity)
- Energetic but not decorative (the form ITSELF is the message)
- Would work on a revolutionary/action poster
- Scores **7/10+** on visual gravity test (estimated)

**Why It Works:**
```
Visual Gravity Test:
"Would this appear on a protest banner or revolutionary flag?"
Answer: YES ✅ — Lightning = action, energy, revolutionary change

Monolithic Form Check:
"Is this a single unified shape?"
Answer: YES ✅ — Pure zigzag, no separate decorative elements

Political Meaning:
"What does this symbol communicate?"
Answer: SPEED, ENERGY, ACTION, INNOVATION — all politically meaningful
```

**Design Comparison:**

| Dimension | Star+Sparks | Lightning |
|-----------|------------|-----------|
| **Symbolism** | Generic innovation + sparkles | Energy, speed, action, innovation |
| **Political Meaning** | NONE (corporate) | ✅ YES (revolutionary energy) |
| **Monolithic Form** | ❌ NO (2 separate paths) | ✅ YES (single path) |
| **Visual Gravity** | 2/10 | 7/10+ |
| **ViewBox Fill** | ~50% (star) + small sparks | ~75% (bold zigzag) |
| **Decorative Ornaments** | ❌ YES (spark lines) | ✅ NO (pure form) |
| **Banner Test** | ❌ NO | ✅ YES |
| **Readability (28px)** | ✅ YES (if you ignore sparks) | ✅ YES (bold and clear) |

## Key Design Decisions

### Why Leaf Over House?
- **House** = Generic utility (no meaning)
- **Leaf** = Growth, life, personal development (politically meaningful)
- **Result:** Transforms "PESSOAL" from a location indicator to a growth symbol

### Why Hammer (Refined)?
- **Round 2 Hammer** was already good (8/10)
- **Minor refinements** increase boldness without changing concept
- **Result:** Strengthens the most successful icon, ready to ship

### Why Lightning Over Star+Sparks?
- **Star+Sparks** = Decorative ornament disease (SaaS app icon energy)
- **Lightning** = Bold monolithic form with energetic meaning
- **Result:** Replaces decorative fluff with a powerful, unified symbol
