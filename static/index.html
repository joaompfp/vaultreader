Vaults

All vaults found under the mounted /vaults directory. Click any to open.

Writable paths are configured in appdata/config.json via the rw_paths array. A vault's name (e.g. pessoal) makes the whole vault writable; a path (e.g. pessoal/agents/hermes/skills) opens up just that subtree.

Container

Docker will bring it back up automatically. Page will reload after 4s.
